Run HPC applications directly on your hardware

The size and the complexity of future supercomputers will increase on its path to ExaScale systems and beyond. The system software must be improved to handle these complex systems and to simplify the development of scalable applications. HermitCore is a unikernel operating system for high-performance computing. It extends the multi-kernel approach (like http://www.barrelfish.org) and combines it with unikernel features while providing better programmability and scalability for hierarchical systems, which based on multiple cluster-on-a-chip processors.
HermitCore is designed for KVM/Linux but also for bare-metal environments.

It's a prototype - be patient!

HermitCore is just intitiated and currently not production ready. Please use it carefully.

It's a research project

HermitCore is the result of a research project at RWTH Aachen Univeristy (www.os.rwth-aachen.de).
